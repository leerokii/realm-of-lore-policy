Privacy Policy

Last updated: June 16, 2025

Realm of Lore is a creative writing assistant Chrome Extension built for fantasy writers, RPG fans, and worldbuilders. It uses the OpenAI API to generate text based on your input.

What Data We Collect

This extension does not collect, store, or share any personally identifiable information. All prompts are processed locally and sent only to the OpenAI API for response generation.

Use of the OpenAI API

Prompts entered are transmitted securely to OpenAI’s servers via HTTPS to generate a response. No input data is stored or analyzed by the extension itself.

Third-Party Services

This extension relies on OpenAI’s API. Please refer to OpenAI’s Privacy Policy for more details.

Changes to this Policy

We may update this policy as needed. Any changes will be posted on this page.
